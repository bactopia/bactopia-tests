# Download Sylph database

The sylph database is too large to be put in this repo. Instead, you will need to download
it yourself.

To get the database, run the following command:

```{bash}
# For latest URL visit: https://github.com/bluenote-1577/sylph/wiki/Pre%E2%80%90built-databases
# GTDB r220 database (113,104 species representative genomes) - 24th April, 2024
# -c 1000 more efficient, less sensitive database (2.6 GB)
wget -O sylph.syldb http://faust.compbio.cs.cmu.edu/sylph-stuff/gtdb-r220-c1000-dbv1.syldb
```
